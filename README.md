# SQL Practice Notebooks (2026)

This repository contains a series of Jupyter notebooks dedicated to strengthening data analysis skills through hands-on exercises in Python, Pandas, Data Visualization, and SQL database management.

---

##Project Overview

This exercise is structured as a comprehensive refresher covering essential data manipulation and querying techniques :

* **SQL & Relational Databases**: Building schema structures, applying constraints, executing complex queries (`JOIN`, `LEFT JOIN`, `GROUP BY`, `HAVING`, subqueries), and ensuring data integrity.
* **Data Processing with Pandas**: Cleaning, filtering, transforming, and aggregating datasets.
* **Data Visualization**: Creating clean plots and graphical representations using `Matplotlib`.

---

##Repository Structure

`Projet_01/` – Directory containing project resources and assets.
* `Fichier_propre.ipynb` – Clean, fully executed Jupyter Notebook containing all completed exercises and solutions.
* `Installation.ipynb` – Setup guide for dependencies and environment configuration.
* `README.md` – Project overview and documentation.
* `pyproject.toml` & `.python-version` – Configuration files for Python version management and project dependencies.
* `.gitignore` – Specifies intentionally untracked files (such as local virtual environments) to ignore.
